JavaScript Closure with Examples

Closure:

Closure is a function that can remember the variable from its outer function, even after the outer function has executed.

A closure allows a function to access from its outer scope, even after the outer scope finished the execution.

Usefulness of Closure

  • You can keep the variables private without exposing them
  • You can stop variable pollution
  • You can create a function factory
  • You can keep a variable alive between multiple calls
